# Grid Bot — Flujo en simple

Goal: explicar cómo funciona el grid bot del usuario en un video corto, en español rioplatense, sin jerga, para alguien que no es trader.
Audience: amigos, familia, cualquier persona curiosa que NO opera mercados.
Duration: ~160s (2:40)
Format: 1920x1080 landscape
Style: Dark mode tipo Bitget — fondo negro/grafito, acentos verde (compra/ganancia) y rojo (venta/pérdida). Tipografía grande. Animaciones suaves con GSAP. Sin TTS, todo en pantalla.

## Escenas

1. (0-10s) El problema — el precio sube y baja todo el día.
2. (10-30s) La idea de grid — red de pesca de órdenes.
3. (30-60s) Un ciclo paso a paso — compra a $0.20, TP a $0.2034, gana, repite.
4. (60-80s) Hedge mode — gana en ambas direcciones a la vez.
5. (80-110s) Defensa contra TP fallido — 3 fases: tranqui, ajusta, fuerza salida.
6. (110-130s) El supervisor — Claude revisa cada 2h, solo recomienda.
7. (130-150s) Estado actual — 5 pares, equity $57.66, +$2.33 hoy.
8. (150-160s) Cierre — "Trading mientras dormís".
