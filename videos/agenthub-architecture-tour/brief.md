# AgentHub Architecture Tour

Objetivo: explicar visualmente cómo funciona AgentHub usando diagramas animados y capturas reales de la UI local.

Estructura:
1. Intro: un cerebro, dos superficies.
2. Captura real del chat y flujo de entrada.
3. Diagrama runtime: daemon Go, SQLite, engines, MCP tools.
4. Operación: System UI, servicios, túneles, cronjobs.
5. Proyectos/skills: workflows aislados por proyecto.
6. Salidas: texto, media clickeable, WhatsApp/outbox.

Estética: Cyberpunk HUD de AgentHub (`DESIGN.md`): fondo navy, cyan/magenta/lime, paneles recortados, líneas de circuito.
